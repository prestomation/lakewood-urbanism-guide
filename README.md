# Lakewood Urbanism Guide

A practical reference guide for urbanists and advocates in **Lakewood, Ohio** — the most walkable and densely populated city in the state.

**Live site:** https://lakewood.urbanism-guide.com/

## What This Is

This Hugo static site helps residents, advocates, and newcomers understand the policies, terminology, and history behind Lakewood's urban character. It covers:

- **Quick Start** — How Lakewood's government works
- **Glossary** — Plain-language definitions for planning and zoning terms
- **Guides** — In-depth coverage of walkability, bike infrastructure, transit, and housing
- **Timeline** — Key events in Lakewood's urban development
- **Data** — Public data sources and dashboards
- **Blog** — Analysis and commentary

## Local Development

```bash
# Install Hugo (extended version 0.147.0+)
# https://gohugo.io/installation/

# Clone with submodules
git clone --recurse-submodules https://github.com/prestomation/lakewood-urbanism-guide.git
cd lakewood-urbanism-guide

# Start development server
hugo server -D
```

The site will be available at http://localhost:1313/
